# sotasato-www

My portfolio page.

<https://group-mmm.org/~sotasato/>

## Usage

To build and launch a local server with port :8000,

```console
$ make build
$ make serve
```

To deploy the `public` directory,

```console
$ git remote add deploy https://<yourrepourl>.git
$ make deploy
```

## Requirements

- pandoc >= 2.14
- python >= 3.7
